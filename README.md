# boomerang-vue

> 失物招领前端vue实现

## 开始开发

``` bash
# 安装依赖
npm install

# 在 localhost:3000 建立热更新服务
npm run dev

# 打包生产环境的代码
npm run build

# 打包生产环境并输出报告
npm run build --report
```
